## v1 Requirements

Requirements for initial release. Each mapped to roadmap phases.

### Foundation & Navigation (7/7)

- [x] **NAV-01**: User can switch between Arabic and English languages
- [x] **NAV-02**: Layout automatically switches between RTL (Arabic) and LTR (English)
- [x] **NAV-03**: Smooth animated transitions between pages
- [x] **NAV-04**: Custom loading animation with Jerash logo on initial load
- [x] **NAV-05**: Mobile hamburger menu with animated open/close
- [x] **NAV-06**: Header with navigation links to all main pages
- [x] **NAV-07**: Footer with PO Box, contact info, and office locations

### Homepage (9/9)

- [x] **HOME-01**: Hero section with image slideshow and cinematic animations
- [x] **HOME-02**: Vision statement section
- [x] **HOME-03**: Mission statement section
- [x] **HOME-04**: Company values section with visual presentation
- [x] **HOME-05**: Management philosophy section
- [x] **HOME-06**: Animated statistics counters (years, projects, employees)
- [x] **HOME-07**: Partners preview section with SLB logo
- [x] **HOME-08**: Joint Ventures preview section with Kweti reference
- [x] **HOME-09**: Services overview with links to full services page

### Services Page (5/5)

- [x] **SERV-01**: Services organized into categories (Production, Wireline, Consultancy, Other)
- [x] **SERV-02**: Interactive service cards that expand on click for details
- [x] **SERV-03**: Filter functionality to show services by category
- [x] **SERV-04**: All 26 services from company profile included
- [x] **SERV-05**: Bilingual service names and descriptions

### HSE Page (Crown Jewel) (5/5)

- [x] **HSE-01**: HSE commitment statement prominently displayed
- [x] **HSE-02**: All 10 HSE commitment points from company profile
- [x] **HSE-03**: Safety statistics/metrics with visual presentation
- [x] **HSE-04**: HSE-related field images (PPE, safety signs, team with gear)
- [x] **HSE-05**: Bold visual design emphasizing importance of safety culture

### Gallery Page (4/4)

- [x] **GALL-01**: Masonry grid layout for all 26 images
- [x] **GALL-02**: Lightbox view on image click with navigation
- [x] **GALL-03**: Lazy loading for performance optimization
- [x] **GALL-04**: Images renamed and organized with proper metadata

### Contact Page (4/4)

- [x] **CONT-01**: General inquiry form (name, email, phone, message, department)
- [x] **CONT-02**: Display of all 3 office locations (Basrah, Erbil, Baghdad)
- [x] **CONT-03**: Clickable phone number and email links
- [x] **CONT-04**: Form validation with error messages in both languages

### Careers Page (4/5)

- [x] **CARE-01**: CV upload form with drag-and-drop file upload
- [x] **CARE-02**: Basic applicant info fields (name, email, phone, position interest)
- [~] **CARE-03**: CV submission sends email notification AND stores in database *(frontend ready, backend deferred to v2)*
- [x] **CARE-04**: File type and size validation for uploads
- [x] **CARE-05**: Success/error feedback in both languages

### Partners & Joint Ventures (4/4)

- [x] **PART-01**: Partners page with logos and descriptions (SLB initial)
- [x] **PART-02**: Joint Ventures page with company details (Kweti initial)
- [x] **PART-03**: Data structure ready for future CMS/API integration
- [x] **PART-04**: Bilingual partner and JV content

### Visual & Animation (5/5)

- [x] **ANIM-01**: Scroll-triggered reveal animations throughout site
- [x] **ANIM-02**: Hover effects on interactive elements
- [x] **ANIM-03**: Bold cinematic animations on hero and key sections
- [x] **ANIM-04**: Respect prefers-reduced-motion for accessibility
- [x] **ANIM-05**: Only animate transform/opacity for 60fps performance

### Technical & SEO (7/7)

- [x] **TECH-01**: Mobile responsive design across all pages
- [x] **TECH-02**: SEO meta tags for both Arabic and English
- [x] **TECH-03**: Proper hreflang implementation for bilingual SEO
- [x] **TECH-04**: Image optimization (WebP, responsive srcset)
- [x] **TECH-05**: Page load under 3 seconds on average connection
- [x] **TECH-06**: Accessible color contrast ratios
- [x] **TECH-07**: Semantic HTML structure
