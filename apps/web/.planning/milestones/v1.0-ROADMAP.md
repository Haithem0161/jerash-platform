# Milestone v1.0: MVP

**Status:** ✅ SHIPPED 2026-01-23
**Phases:** 1-9
**Total Plans:** 24

## Overview

This milestone delivered a bilingual (Arabic/English) corporate website for Jerash For Oil Field Services, progressing from RTL-aware foundation through animated homepage sections, content pages (with HSE as crown jewel), image gallery, and forms. The 9-phase structure ensured RTL conventions and animation patterns were established before content development, preventing costly rewrites. Each phase delivered a verifiable, complete capability.

## Milestone Summary

**Key Decisions:**
- Ken Burns hero effect with CSS keyframes for performance
- Single-URL multilingual (client-side toggle, not /ar/ /en/ paths)
- PWA with service worker for offline capability
- Code splitting by page for optimal loading
- jsx-a11y as warnings, not errors

**Issues Resolved:**
- RTL layout switching (dir attribute on html)
- Parallax performance (spring easing, GPU-accelerated)
- Form validation i18n (translation keys, not strings)
- a11y compliance (skip links, focus styles, semantic HTML)

**Issues Deferred:**
- Backend API for form submissions (v2)
- Admin panel for content management (v2)
- Real contact info (pending client)
- Partner/JV logos (pending client assets)

**Technical Debt Incurred:**
- 4 jsx-a11y warnings (onClick on non-interactive elements)
- Placeholder images for HSE (1px JPEGs)
- Contact info placeholders (TODO comments)
- LinkedIn URL placeholder
